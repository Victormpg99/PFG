# PFG
PFG sobre el procesamiento de datos de automóviles 
Para probar la herramienta en tu equipo:

Asegúrate de tener las librerías instaladas: pip install -r requirements.txt

Una vez ejecutado dashboard.py, se debe ejecutar el comando desde la raíz del proyecto: streamlit run src/dashboard.py

Se abrirá automáticamente una pestaña en tu navegador donde podrás cargar cualquiera de los archivos CSV adjuntos.
